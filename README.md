#Час кода для гирлянды

####Структура каталогов

```
    base/                основное приложение
        style/           внешний вид страниц
        level/           HTML-страницы с уровнями
        config/          конфигурации уровней
        proc/            обработка ошибок
        common/          общие функции
        sol/             готовые решения заданий
        i18n/            строковые константы, тексты переводов
        vendor/          библиотеки сторонних разработчиков

    sandbox/             разработка и тестирование новых функций
        style/           внешний вид страниц    
        vendor/          библиотеки сторонних разработчиков
```
